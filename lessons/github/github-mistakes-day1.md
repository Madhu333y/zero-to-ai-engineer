# GitHub Mistakes — Day 1

*March 17, 2026 — First day setting up the repository*

---

### Mistake 1 — Created folder as a file
**What happened:**
Typed "journey" as filename
GitHub created a file not a folder

**Fix:**
Delete the file first
Recreate using full path:
journey/month-01-python-runtime-terminal/week-01.md

---

### Mistake 2 — Pasted instructions inside file
**What happened:**
Copied everything including instructions
into the file content

**Fix:**
Ctrl + A → Delete → paste only the content

---

### Mistake 3 — File exists error
**What happened:**
Error: file exists where trying to
create subdirectory

**Fix:**
Delete existing file first
Then recreate with correct path

---

### Mistake 4 — Journey folder showing full path
**What happened:**
Sidebar showed full path instead of
just "journey"

**Fix:**
Add README.md directly inside journey folder

---

### Mistake 5 — Default commit messages
**What happened:**
Left GitHub auto-generated messages

**Fix:**
Always replace with:
action: what you did

---

### Mistake 6 — Old commits permanent
**Lesson:**
Commits locked forever once made
Fix forward — never back
